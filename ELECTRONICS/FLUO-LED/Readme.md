## FLUO-LED (Arduino NANO)

For exciting fluorophores we used power-led's with an appropriate spectrum.

Our configuration for GFP is:

	- (Royal-Blue) Power LED
	- LED-Collimator
	- COMAR 480 dichroic
	- COMAR 520 Emission Filter
